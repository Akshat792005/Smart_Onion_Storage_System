# 🧅 Smart Onion Storage System  

## 📌 Overview  
The Smart Onion Storage System is an IoT-based solution designed to monitor and maintain optimal storage conditions for onions. It reduces post-harvest losses by controlling environmental factors such as temperature, humidity, and ventilation.

---

## 🎯 Objective  
- Prevent spoilage of onions during storage  
- Maintain ideal temperature and humidity levels  
- Automate ventilation and monitoring  
- Provide real-time alerts and data tracking  

---

## ⚙️ Features  
- Temperature Monitoring  
- Humidity Monitoring  
- Automatic Ventilation System  
- Alert System (Buzzer / Notification)  
- IoT-based Remote Monitoring  
- Real-time Data Display  

---

## 🛠️ Components Used  
- Microcontroller (Arduino / NodeMCU / ESP8266)  
- DHT11 / DHT22 Temperature & Humidity Sensor  
- Fan / Ventilation System  
- Relay Module  
- LCD Display (16x2)  
- Buzzer  
- Power Supply  
- Connecting Wires  

---

## 🔧 Working Principle  
1. The sensor continuously measures temperature and humidity inside the storage system.  
2. Data is sent to the microcontroller.  
3. If conditions exceed threshold limits:  
   - Fan is activated for ventilation  
   - Alerts are triggered  
4. Data can be monitored on display or mobile app.  

---

## 📐 System Architecture  
- Input: Temperature & Humidity Sensors  
- Processing: Microcontroller  
- Output: Fan, Buzzer, Display  
- Connectivity: IoT (Optional Mobile App / Cloud)
